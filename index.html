SwissVote

Mapping the Polis API

Overview and instructions.

This web app visualizes votation results on a cartogram of the cantons of Switzerland. You search for any federal vote dating back to the year 1900 by using the autocomplete form below (the gold box).

For a more visually interesting example, try typing in "test" below, and selecting the available option.

About this project.

This app is a work-in progress, and it was created at the 2018 SSR HackDays in Geneva, by learners and instructors of the EPFL Extension School.

All of the data was retrieved from the Polis API, which is the Political Information System that is maintained by SRG SSR. Polis provides results for votations and elections in Switzerland and other countries back to 1900.

The following features are being worked-on, and will be coming soon:

  • Polis API integration & support for French, and Italian votation records.
  • Tool-tips for detailed viewing of vote results.

You can find all of the source-code from this project on the EPFL Extension School's GitHub page.
